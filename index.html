Quicken - Surf faster than your mouse can handle

Details

Google webstore link: https://chrome.google.com/webstore/detail/quicken/bcnfaiaknkbbcmpeedkjjjcdmjlgbafo

A Google Chrome extension giving quick access to the 'next page' button in various sites.

You can use a keyboard shortcut - Ctrl+Right (right arrow).

Supported Sites

  • Google search (www.google.com)
  • Ars Technica (www.arstechnica.com)
  • blogspot
  • wordpress
  • tumblr
  • and many more ...

Install

Install the extension using 'Developer mode' in the extensions page. After enabling the 'Developer mode', select 'Load unpacked extension...' and select the directory where you've saved the files.
